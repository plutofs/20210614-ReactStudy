실행법

1. 기초적인 리액트 실행에 필요한 것들을 받아놓는다.
2. VSCode를 실행한다
3. 이 git의 클론을 생성한다.
4. Vscode의 터미널을 실행해 git clone 링크  로 클론을 생성한다.
5. 다됬다면, cd my-app 을 통해 실제 react 폴더에 접근한다
6. npm install을 통해 데이터를 다운받는다
7. npm start한다. 끝.
